#### Non-Heated Util Mod 1.0.1 for Minecraft 1.12.2 ####


# 前提Ver&Mod

・Minecraft1.12.2
・MincraftForge 1.12.2-14.23.3.2768 以上

# 使用方法
・MincraftForgeの導入後に作成されるmodsフォルダの中に、HeatAndClimateLibとともに、DLしたファイルを.jarファイルのまま置いてください。

# 配布DL先
 投稿フォーラム：http://forum.minecraftuser.jp/viewtopic.php?f=13&t=17657
 URL：http://defeatedcrow.jp/modwiki/HeatAndClimate


# 概要

　HeatAndClimateの要素のうち、一部の便利ブロック・エンティティーを取り出した小物Modです。
　主に、旧鉄鎖ModやFluidityから移植してきた、AMTに由来しない要素を中心に抜き出しています。

　NoSupport、NoRequestの単発Modですので、追加要望などは受け付けません。


# Configについて #############################################################################

　レシピに必要な鋼鉄(Steel)のインゴットとギアを追加していますが、
　これらのレシピや辞書登録をコンフィグでオフに出来ます。
　他ModのSteelがすでにあり、不要の場合は、コンフィグでOFFにしてください。

# 詳細 #############################################################################

・蛇口

　ブロックの下面・側面に取り付けられる液体輸送管です。
　重力を利用するため、動力は不要ですが、出口は下向きしかありません。
　右クリックで開閉できます。

　液体タンクとの間に別のブロックを1ブロック挟んでいても、ブロック越しに液体を輸送できます。
　(景観対策で蛇口を壁に取り付けたいケースに対応させるため。)

・無限水源蛇口

　ブロックの下面・側面に取り付けられる液体輸送管です。
　取り付け方向は動作に影響しません。（景観用途）
　右クリックで開閉できます。

　真下にある液体コンテナに、1000mB/5tで水を送り続ける装置です。
　バルブが閉になると止まります。

・IBCコンテナ

　容量が128バケツ分の液体タンクです。ドロップ時の中身保持機能は健在です。
　見た目は透明になって、中身は見やすくなりました。

　液体容器Itemとしても振る舞います。

・フィルターホッパー

　スロットにアイテムが1個だけ残る状態で移送を行います。
　それ以外はバニラホッパーとだいたい同様の仕組みです。

　各スロットにアイテムを1個残すため、予め入れておいたアイテム以外を通さないフィルターのように使うことができます。
　仕様上、スタックできないものを流すと詰まります。

・液体ホッパー

　1.7のFluidityFoodstuffsにあったものの簡易版。
　液体を上から下に移送できるホッパーです。蛇口と違い、液体入りのアイテムもやりとりすることができます。
　横方向には自動搬入・搬出能力がないため、バニラホッパーやベルトコンベアと組み合わせるのを推奨します。

　-真上の液体ブロックを回収する
　-真上の液体タンクからの自動搬入、真下の液体タンクへの自動搬出
　-真上のインベントリから液体容器を自動搬入、真下のインベントリへ処理済み液体容器を自動搬出
　-液体容器を充填・空にする

　また、真上に牛がいるとミルクを搾り取る機能もあります。
　(他のModが液体ミルクを追加している場合のみ)

　RS信号により停止します。

・遠隔モニター(RS、コンパレータ出力)

　離れた位置のブロックの状況を確認できるモニターです。
　RSおよびコンパレータのモニターランプは、ONのときにRS信号を発生します。

　使い方：
　　アイテム状態で手に持って、対象の座標をスニーク+右クリックで座標登録です。
　　その後、同じディメンジョン内に設置しておけば稼働します。
　　距離が離れている場合はチャンクローダーが必要です。

・鋼鉄のはしご
　　支えがなくても自立できるはしごです。
　　同じブロックを手に持って右クリックすることで、簡単に伸長できます。

・花瓶
　　これはブロックではなく、エンティティです。そのため、向きや座標をブロックよりも自由に決められます。
　　花や草などの植物をスニーク右クリックで投入すれば飾ることができます。

・モブをダメにするクッション
　　ブロックではなくエンティティです。
　　接触したモブを強引に引き寄せるクッションです。
　　プレイヤーも座れます。（既に座っているモブを追い出します。）

・メタルチェスト
　中身が入ったまま持ち運びできるチェストです。
　爆発に対しても高い耐性があります。
　ROCK属性なので、つるはしがないと壊せない点に注意。

・マグネットチェスト
　5x5x5ブロック範囲のドロップアイテムを集める機能があります。

・村人チェスト
　近くにいる村人のインベントリがいっぱいになると、アイテムを預かってインベントリに空きを作ります。
　村人はインベントリが一杯になると農作業ができなくなるので、それを補助するために作ったブロックです。

・足場ブロック
　建設に便利な足場です。
　- はしごのように側面を上り下りできます。
　- 設置後、ブロックを持って右クリックすると、設置済みの足場を手持ちのブロックで一括置換します。

・エメラルドの感圧板
　プレイヤーと飼いならされたモブにのみ反応する感圧板です。
